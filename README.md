# Dashboard-Kehadiran-Karyawan
Proyek ini berfokus pada pemanfaatan metode Data Science, khususnya analisis data 
eksploratif dan visualisasi data. Data yang digunakan adalah data kehadiran karyawan, yang 
dianalisis untuk mengidentifikasi pola dan tren yang muncul. Selain analisis deskriptif, proyek ini 
juga menggunakan metode clustering untuk mengelompokkan karyawan ke dalam kategori tertentu 
berdasarkan berbagai indikator seperti lama kerja, keterlambatan, pulang cepat dan durasi istirahat. 
Pengelompokan ini bertujuan untuk memudahkan identifikasi kelompok karyawan dengan 
karakteristik kedisiplinan yang berbeda. 
Hasil analisis disajikan dalam bentuk visualisasi berupa dashboard interaktif yang 
dikembangkan menggunakan Streamlit. Dashboard ini dibuat untuk 
memudahkan dalam memahami pola keterlambatan dan kedisiplinan serta menganalisis jam kerja 
dan istirahat setiap karyawan secara lebih komprehensif dan berbasis data.
